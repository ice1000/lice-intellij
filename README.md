## Lice IntelliJ

This is a IDE for [the Lice programming language](https://github.com/lice-lang/lice),
based on the intellij platform.

It provides syntax highlighting and some structural editing tools.

## Progress

+ [X] Live templates
+ [X] Syntax viewer
+ [X] Semantic viewer
+ [X] Commenter
+ [X] Create new file
+ [X] Parser and lexer
